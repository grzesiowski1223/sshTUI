# sshTUI
TUI ssh wrapper for bash 3+

## Installing sshTUI
```bash <(curl -s https://raw.githubusercontent.com/grzesiowski1223/sshtui/main/install_sshtui.sh)```

## Dependency
This project depends on fzf (fuzzy finder)

##### Debian/Ubuntu:
```sudo apt install fzf```

##### Arch Linux (and all the deriviatives)
```sudo pacman -S fzf```
